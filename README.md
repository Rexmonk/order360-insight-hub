# mapi-pom-pommes-ui

## Development

### Use in DevContainer

Prepare your local copy

```
npm i
```

(Optional) Run `npm ci` to clean the node_modules folder and reload all.

### Run it local

```
npm run local
```

### Regenerate OpenApi Model

(use DevContainer)

```
npm run codegen
```

### Run tests

```
npm run test
```
